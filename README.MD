
# Todo Typescript graphql

### Backend
To start the api server:

1. Navigate to the `api` folder
2. Run `npm i`
3. Run `npm run build`
4. Run `npm run dev`

To stop the server:
Focus on the terminal window where you started it and press `CTRL + C`.

### Frontend

1. Navigate to the `ui` folder
2. Run `npm i`
3. Run `npm run build`
4. Run `npm run dev`
